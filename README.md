# AI Notes & PDF Summarizer

A GenAI project that generates notes and summaries from text and PDF files.

## Features
- Text summarization
- PDF summarization
- Uses LLM API (OpenRouter / Gemini)

## Tech Stack
- Python
- Flask
- OpenRouter API
- PyPDF2

## Setup
1. Clone repo
2. Install requirements:
   pip install -r requirements.txt
3. Add .env file:
   OPENROUTER_API_KEY=your_key
4. Run:
   python app.py
